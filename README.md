# xiaoeyun's OpenToonz Animations
This repository contains a collection of animations created by xiaoeyun using OpenToonz. 

## About assets
Please see [releases](https://github.com/xiao-e-yun/animations/releases) for the assets used in these animations.  
And you need replace `scenes\[scene].tnz` to correct path of the scene file in the assets. (e.g. `S:\\images\\animations\\mayu-1`)
